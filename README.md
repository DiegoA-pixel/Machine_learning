Titulo: Proyecto para predecir el precio de alojamiento en Airbnb con lo aprendido en machine learning.
A partir del dataset proporcionado se realizo el proceso completo requerido de limpieza preprocesamiento, seleccion de variables
y comparacion de modelos predictivos, obteniendo como conclusion que el analisis por random forest obtiene un desempeño ligeramente
superior a la regresion lineal, ademas de que las variables relacionadas con el tamaño del alojamiento son las que mas influyen en 
el precio.
